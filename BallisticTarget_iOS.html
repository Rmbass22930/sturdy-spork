Saving Targets Locally

  1. Open the Files app → On My iPad → tap the “…” menu → New Folder → name it BallisticTargetOutput.
  2. In Safari, after tapping Download PNG, tap the download icon → the file → Share → Save to Files.
  3. Select On My iPad > BallisticTargetOutput and tap Save. Safari remembers the last location, so future saves go there automatically.
  4. Need to move files back to desktop? Plug in a portable SSD or use AirDrop/Email from the Files app.
