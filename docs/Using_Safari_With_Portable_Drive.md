How to Open BallisticTarget HTML Files in Safari (iPad)
====================================================

Use this handout to walk someone through loading the BallisticTarget web tools from a USB/SSD into Safari.

1. **Connect the drive**
   - Plug the portable SSD/USB drive into the iPad (USB‑C cable or Lightning adapter).

2. **Launch Files**
   - Open the **Files** app.
   - Tap **Browse** (bottom).
   - Under **Locations**, tap the drive name (e.g., “SSD_NAME”).

3. **Find the HTML file**
   - Navigate to the folder containing BallisticTarget_iOS.html or EnvironmentalsGeo_iOS.html.
   - Tap the HTML file once to preview it.

4. **Open in Safari**
   - Tap the **Share** icon (square with arrow) in the preview.
   - Choose **Open in Safari**.
   - Safari loads the file directly from the drive; the app is now ready to use offline.

5. **Optional: Add to Home Screen**
   - In Safari, tap the Share icon → **Add to Home Screen** for an app-like icon.

6. **Saving output back to the drive**
   - After downloading a PNG/config.json inside Safari, tap the download icon (top-right) → **Downloads** → select the file → **Save to Files** → pick the SSD.

7. **Disconnect safely**
   - Return to Files, tap the eject icon next to the drive name before unplugging.

Tips
----
- Keep the SSD plugged in while using the HTML; if you unplug it, Safari can’t reload the page.
- You can copy the HTML to iCloud Drive or “On My iPad” if you prefer to run without the SSD.
- All logic runs locally—no installer, no App Store approval needed.
